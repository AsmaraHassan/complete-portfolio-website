Project 4

Vertical lines

Create a Vertical Black and White Line in Photoshop Open Photoshop: Start Photoshop and create a new document with your desired dimensions. Choose the Line Tool: In the toolbar on the left side, select the Line Tool (U). If it is not visible, right-click the shape tools until the Line Tool appears. Set the Line Color: In the top menu, find the color picker and set the Foreground Color to black. You can use a pure black color by entering the HEX code #000000 in the color picker window. Set the Stroke Weight: On the top properties bar, set the Stroke Weight to define the thickness of the line. A value like 3-5px is good for basic lines, but adjust as needed.
